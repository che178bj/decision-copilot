# Decision Co-Pilot

您的 AI 決策小幫手 - 幫您分析重大決定，找出盲點，做出更好的選擇。

## ✨ 功能特色

- 📊 **利弊分析** - 系統化列出決策的優點與缺點
- ❓ **關鍵問題** - 問出您可能忽略的重要問題
- 🎯 **建議輸出** - 基於分析給出建議參考
- 💾 **決策記錄** - 本地端儲存所有決策紀錄
- 🔔 **回顧提醒** - 決策後定時提醒您回顧結果

## 📦 安裝方式

```bash
pip install decision-copilot
```

## 🚀 使用方式

### 啟動決策輔助

```bash
decision-copilot
```

### 列出所有決策

```bash
decision-copilot list
```

### 回顧特定決策

```bash
decision-copilot review <id>
```

### 更新決策狀態

```bash
decision-copilot status <id> <status>
```

## 💬 使用範例

```
🔮 Decision Co-Pilot - 您的決策小幫手

請描述您面臨的決定：我要不要接受這份新工作？

❓ 請回答幾個問題：

1. 這份工作的薪水比目前高多少？（%）
→ 用戶輸入：30%

2. 這是您有興趣的產業嗎？（是/普通/否）
→ 用戶輸入：是

3. 您了解未來團隊文化嗎？（了解/不了解）
→ 用戶輸入：了解一點

4. 這份工作會讓您通勤時間增加嗎？（增加/不變/減少）
→ 用戶輸入：增加30分鐘

🎯 最終建議：【接受】

📊 建議說明：
基於您的回答，優點權重較高，建議接受。
```
